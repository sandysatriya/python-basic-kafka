# python-basic-kafka

Kafka implementation for python

## Installation

Create virtual environment.

```bash
python3 -m venv venv
```
Activate that virtual env.

```bash
source venv/bin/activate
```
Install requirements

```bash
pip install -r requirements.txt
```
